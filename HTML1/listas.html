Las listas en HTML son esenciales para organizar información en tus páginas web. HTML ofrece tres tipos principales: <ul> (desordenadas), <ol> (ordenadas) y <dl> (de definición). Exploremos cada una con ejemplos y detalles.

Listas Desordenadas (<ul>)

Las listas desordenadas (unordered lists) se usan para elementos sin un orden específico. Se representan con viñetas.

Ejemplo básico:

  • Elemento 1
  • Elemento 2
  • Elemento 3

Atributos:

  • type: Especifica el tipo de viñeta:
    • disc (predeterminado): Círculo relleno
    • circle: Círculo vacío
    • square: Cuadrado relleno

Ejemplo con square:

  • Elemento 1
  • Elemento 2

Ejemplo con disc:

  • Elemento 1
  • Elemento 2

Ejemplo con circle:

  • Elemento 1
  • Elemento 2

Listas Ordenadas (<ol>)

Las listas ordenadas (ordered lists) se usan cuando el orden de los elementos es importante. Se representan con números o letras secuenciales.

Ejemplo básico:

  1. Primer paso
  2. Segundo paso
  3. Tercer paso

Atributos:

  • type: Especifica el tipo de numeración:
    • 1 (predeterminado): Números arábigos
    • a: Letras minúsculas
    • A: Letras mayúsculas
    • i: Números romanos en minúsculas
    • I: Números romanos en mayúsculas
  • start: Especifica el número o letra de inicio.
  • reversed: Invierte el orden de la lista.

Ejemplo con type 1 start 3:

  1. Tercer paso
  2. Cuarto paso

Ejemplo con type a start 3:

  1. Tercer paso
  2. Cuarto paso

Ejemplo con type A start 3:

  1. Tercer paso
  2. Cuarto paso

Ejemplo con type i start 3:

  1. Tercer paso
  2. Cuarto paso

Ejemplo con type I start:

  1. Tercer paso
  2. Cuarto paso

Ejemplo con reversed:

  1. Tercer paso
  2. Segundo paso
  3. Primer paso

Listas de Definición (<dl>)

Las listas de definición (description lists) se usan para presentar términos y sus definiciones. Cada término se define con <dt> (definition term) y su definición con <dd> (definition description).

Ejemplo:

HTML
Lenguaje de marcado para la creación de páginas web.
Lenguaje de marcado para la creación de páginas web.
Lenguaje de marcado para la creación de páginas web.
CSS
Lenguaje de estilos para dar formato a las páginas web.
Lenguaje de estilos para dar formato a las páginas web.
Lenguaje de estilos para dar formato a las páginas web.

No tiene atributos específicos.

Listas Anidadas

Las listas pueden anidarse para crear estructuras más complejas.

Ejemplo:

  • Elemento 1
    • Sub elemento 1
    • Sub elemento 2
  • Elemento 2
