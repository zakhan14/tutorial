¿Qué son las tablas en HTML?

Las tablas en HTML permiten organizar información en filas y columnas.

Estructura de una Tabla

Una tabla en HTML se crea con la etiqueta <table> y se compone de:

  • <tr> - Define una fila.
  • <th> - Define un encabezado de columna.
  • <td> - Define una celda de datos.

Ejemplo de una Tabla Básica

        <table border="1">
            <tr>
                <th>Nombre</th>
                <th>Edad</th>
                <th>Ciudad</th>
            </tr>
            <tr>
                <td>Juan</td>
                <td>25</td>
                <td>Madrid</td>
            </tr>
            <tr>
                <td>María</td>
                <td>30</td>
                <td>Barcelona</td>
            </tr>
        </table>

Aplicando Estilos a la Tabla

Podemos mejorar la apariencia con CSS:

        <style>
            table {
                width: 100%;
                border-collapse: collapse;
            }
            th, td {
                border: 1px solid black;
                padding: 10px;
                text-align: left;
            }
            th {
                background-color: #0e569e;
                color: white;
            }
        </style>
