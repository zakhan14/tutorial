Pseudo-Clases Interactivas

INICIO RESUMEN COLORES LENGUAJES ATAJOS DE TECLADO LISTAS FLEXBOX IMAGENES TABLAS MEDIA QUERIES

Las pseudo-clases mejoran la experiencia de usuario al cambiar la apariencia de los elementos según su estado. Es una forma de escribir algoritmos de manera informal, utilizando un lenguaje cercano al humano pero con estructuras lógicas de programación. Su objetivo es facilitar la planificación de programas antes de implementarlos en un lenguaje de programación real. Se usan para definir estilos especiales en ciertos estados de un elemento (ej: :hover, :active, :focus).

  • Inicio de la lista
  • Elemento con efecto hover
  • Elemento activo al hacer clic
  • Elemento impar con color especial
  • Última opción con estilo único

los codigos utilizados son los siguientes:

ul { list-style: none; padding: 0; }
li { padding: 10px; margin: 5px; background-color: #fff; border: 2px solid #555; transition: 0.3s; }
li:hover { background-color: #ffc107; transform: scale(1.1); }
li:first-child { background-color: #8bc34a; font-weight: bold; }
li:last-child { background-color: #e91e63; font-style: italic; }
li:nth-child(odd) { background-color: #03a9f4; }
li:active { background-color: #ff5722; }
