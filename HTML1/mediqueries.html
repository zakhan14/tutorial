Introducción a Media Queries

Las Media Queries en CSS permiten aplicar estilos según las características del dispositivo, como el tamaño de la pantalla.

Ejemplo de Media Query para móviles

@media screen and (max-width: 600px) { body { background-color: #ffebee; } }

Este código cambia el color de fondo cuando la pantalla es menor a 600px.

Ejemplo de Media Query para tablets

@media screen and (min-width: 601px) and (max-width: 1024px) { body { background-color: #e3f2fd; } }

Este código cambia el color de fondo en pantallas entre 601px y 1024px.
